퓨샷 프롬프팅: 몇 가지 예시를 넣어서 원하는 답변을 유도가능.

멀티턴 : messages에 지금까지의 대화내역을 이어서 추가하고 보낸다. 그래야 대화한 내용을 기억할 수 있다.

스트림릿 : 웹 기반 UI로 구현하게 해주는 프레임워크

Hugging Face : AI 모델을 공유하는 플랫폼 (모델의 GitHub 같은 곳)

펑션 콜링 : 모델에게 함수와 그에 관한 설명을 제공하고 상황에 맞는 특정 함수를 호출하도록 함

LangChain : AI 모델을 활용한 애플리케이션을 쉽게 만들 수 있는 프레임워크

주요 기능
- LCEL (|로 체인 연결): prompt | model | parser
- Tool/Function Calling: AI가 함수를 직접 호출
- RAG: 문서 검색 + AI 답변
- Memory: 대화 기록 관리
- PromptTemplate: 프롬프트 템플릿 관리

LangGraph : LangChain 위에서 동작하는 그래프 기반 워크플로우 프레임워크
- 복잡한 AI 에이전트 시스템 구축 가능 (조건 분기, 루프, 병렬 처리)
- 노드(처리 단계) + 엣지(흐름 연결)로 AI 팀 구성 가능
- 책 쓰는 AI처럼 여러 에이전트가 협업하는 시스템 구현

RAG : 문서를 벡터DB에 저장하고, 질문과 관련된 내용을 검색해서 AI 답변에 활용하는 방식
- 임베딩: 텍스트를 숫자 벡터로 변환 (의미가 비슷하면 벡터도 비슷)
- 벡터DB (Chroma): 임베딩된 벡터를 저장하고 유사도 검색

Ollama : 내 컴퓨터에서 LLM을 실행할 수 있게 해주는 도구
- DeepSeek, Llama 등 다양한 모델 실행 가능
- API 비용 없음, 인터넷 없어도 동작

Tavily : AI에 최적화된 웹 검색 API
- 검색 결과를 AI가 바로 활용하기 좋은 형태로 반환

기타 구현 가능한 기능들

- PDF에서 텍스트 추출 : PyMuPDF

- 음성에서 텍스트 추출 : Whisper

- 이미지 분석 

- 텍스트에서 tts 음성 만들기
